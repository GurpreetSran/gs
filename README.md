# Portfolio using Reactjs

## Technology stack
- React
- React-router
- Matter.js
- React Animations
- Browserify
- NPM
- Bootstrap
- React-imageloader

## How to run 
- ```git clone``` repo and ```cd dir```
- ```npm install```
- ```npm start``` (Starts watchify, which bundles js output in backgroud)
- ```run index.html in any modern browser```
- Note - Deploy using node -v 10
